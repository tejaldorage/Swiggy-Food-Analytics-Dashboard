# Swiggy-Food-Analytics-Dashboard

## 📌 Project Overview
This project is an interactive **Excel Dashboard** created using Swiggy food delivery data. The dashboard provides insights into sales performance, customer ratings, order trends, food categories, and restaurant performance across different cities.

The main objective of this dashboard is to transform raw Swiggy data into meaningful business insights using Excel's data analysis and visualization features.

---

## 📊 Dashboard Features

### KPI Cards
- Total Sales
- Average Ratings
- Average Order Value (AOV)
- Overall Ratings
- Total Orders

### Visualizations
- Monthly Sales Trend
- Daily Sales Analysis
- Top 5 Cities by Sales
- Food Type Distribution (Veg / Non-Veg)
- Top 10 Rated Restaurants
- Top Restaurants by Sales
- India Sales Map

### Interactive Filters (Slicers)
- Month
- Area
- Cuisine
- Restaurant
- City
- State
